# ar 포커 용어·표기 브리프 (번역 착수 전 웹리서치 결과)

> 대상 = **아랍어(ar, MSA 표준)**. 아랍 포커 글로시(ggpoker.com/ar · poker4arabs.com · ar.wikipedia تيكساس هولديم/ترتيب إيدي البوكر · toparabcasinos) 기준. **RTL 언어** — 페이지가 `dir=rtl` 자동 처리(RTL_LOCALES=["ar"]), 본문은 자연스러운 아랍어로만 쓰면 됨(라틴 카드표기·숫자는 브라우저 bidi가 LTR로 처리).

## 용어 (현지 관습 — 영어 음차 우세 + 아랍어 등가)
- **행동**: check → **التمرير/تشيك**, bet → **الرهان**, call → **المجاراة/كول**, raise → **الزيادة/الرفع**, fold → **الطي/فولد**, all-in → **أول إن / المراهنة بكل الرقائق**.
- **스트리트**: Flop → **الفلوب**, Turn → **التيرن**, River → **النهر**, preflop → **ما قبل الفلوب**.
- **족보(음차가 표준, 첫 등장 시 괄호로 영어 병기)**: Royal Flush → **رويال فلاش**, Straight Flush → **ستريت فلاش**, Four of a Kind → **فور أوف آ كايند (رباعية)**, Full House → **فول هاوس**, Flush → **فلاش**, Straight → **ستريت**, Three of a Kind → **ثلاثية**, Two Pair → **زوجان**, One Pair → **زوج**, High Card → **ورقة عالية**.
- **핵심 명사**: Blinds → **البلايندز/الرهانات العمياء** (Small Blind=**الرهان الصغير/سمول بلايند**, Big Blind=**الرهان الكبير/بيغ بلايند**), Showdown → **المواجهة/كشف الأوراق**, Kicker → **الكيكر/الورقة المرجحة**, Pot → **القدر/البوت**, Main pot → **البوت الرئيسي**, Side pot → **البوت الجانبي**, Board/community cards → **الأوراق المشتركة/البورد**, Hole cards → **الأوراق الخاصة**, Pot odds → **احتمالات القدر (بوت أودز)**, Stack → **الرصيد/الستاك**, Buy-in → **الدخول (باي إن)**, Dealer/Button → **الموزع/الزر (BTN)**, Bluff → **الخداع/بلاف**.
- **포지션 약어는 영어 그대로**(첫 등장 시 아랍어 병기): UTG(**تحت المدفع**), CO(**الكت أوف**=버튼 오른쪽 첫 자리), MP, HJ, SB, BB, BTN.

## 표기 (⚠️ 기존 ar 관습 준수)
- **숫자 = 서양 아라비아 숫자(0-9)**, 천단위 **콤마**(,), 소수점 **마침표**(.) → `1,326`, `198,000`, `0.003%`, `2.5 BB`, `2.7:1`. **동부 아라비아 숫자(٠١٢٣) 금지**(기존 ar 글이 1,000/2,000식 서양숫자 사용 = §13 숫자 변환 리스크 없음).
- **카드 표기 라틴 유지**: `A♠`, `K♥`, `10♦`, 보드 그대로(기존 ar 글과 동일). 무늬 기호 ♠♥♦♣ 그대로.
- **화폐**: $ 그대로(§13 보존).
- **문장부호**: 아랍어 쉼표 **،**, 물음표 **؟** 사용 권장. 코드 템플릿 리터럴 안에서 직선 따옴표 " 는 안전.
