# it 포커 용어·표기 브리프 (번역 착수 전 웹리서치 결과)

> 대상 = **이탈리아어(it, Italiano)**. 이탈리아 포커 글로시(pokerstars.it · sisal.it/regole-poker · poker.888casino.it · cardplayer.com/it · assopoker.com · it.wikipedia Punti_del_poker) 기준. 이탈리아는 **규제 온라인 포커 시장**(ADM)·고유 포커 용어 확립. ★**직역 아님 — 현지 맥락 재구성**.

## ⚠️⚠️ 치명 함정 (§13급 — 반드시 주의)
1. **Poker = 포카드(Four of a Kind)!** 이탈리아어에서 "Poker"는 게임 이름이자 **포카드 족보명**. quads=**Poker**. 게임 이름과 헷갈리지 말 것.
2. **Colore = Flush** (프랑스 couleur처럼). ⚠️무늬(suit)도 "seme" 또는 "colore"라 문맥 구분. flush 뜻일 때 족보 Colore.
3. **홀덤 족보 순서 = Full(풀하우스) > Colore(플러시)**. ⚠️ 검색 시 나오는 "이탈리아 변형 포커(Poker all'italiana)에서는 Colore가 Full을 이긴다"는 **다른 게임** — **텍사스 홀덤은 표준 순서(Full > Colore)**를 반드시 따를 것. 이탈리아 변형 순서 절대 쓰지 말 것.
4. **Scala = Straight**, **Scala Reale = Royal/Straight Flush**.

## 용어 (현지 관습)
- **족보 (이탈리아어 표준명, 첫 등장 시 영어 병기)**:
  - Royal Flush → **Scala Reale Massima** (또는 Scala Reale, Royal Flush)
  - Straight Flush → **Scala a Colore** (또는 Scala Reale, Straight Flush)
  - Four of a Kind → **Poker** ⚠️(quads)
  - Full House → **Full**
  - Flush → **Colore** ⚠️(flush)
  - Straight → **Scala**
  - Three of a Kind / set → **Tris**
  - Two Pair → **Doppia Coppia**
  - One Pair → **Coppia**
  - High Card → **Carta Alta**
- **액션**: check → **check / bussare**, bet → **puntare / puntata**, call → **chiamare / vedere**, raise → **rilanciare / rilancio**, fold → **passare / foldare**, all-in → **all-in / tutto**. 첫 등장 후 자연스러운 쪽.
- **블라인드**: Blinds → **bui**, Small Blind → **piccolo buio (SB)**, Big Blind → **grande buio (BB)**. 영어 small blind/big blind·SB/BB 병기 흔함.
- **스트리트/구조**: preflop, flop, turn, river 그대로(이탈리아도 영어 유지). giro di puntate(betting round), board/bordo, carte comuni(community cards), carte personali/hole cards(hole cards).
- **핵심 명사**: card → **carta**, hand(패) → **mano**, hand rankings → **classifica delle mani / punti**, dealer → **mazziere / dealer**, button → **bottone (BTN)**, pot → **piatto**, main pot → **piatto principale**, side pot → **side pot / piatto laterale**, kicker → **kicker**, showdown → **showdown / apertura delle carte**, suit → **seme**, stack → **stack**, chips → **fiches / chips**, Bluff → **bluff**, ante → **ante**.
- **포지션 약어는 영어**(첫 등장 병기): UTG, CO(Cutoff), MP, HJ, SB, BB, BTN.
