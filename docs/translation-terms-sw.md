# sw 포커 용어·표기 브리프 (번역 착수 전 웹리서치 결과)

> 대상 = **스와힐리어(sw, Kiswahili)**. 대상 시장 = 동아프리카(케냐·탄자니아·우간다·콩고DR 등, 2억+ 화자). ★**중요**: 스와힐리에는 **확립된 포커 전문 어휘가 사실상 없음**(웹리서치 결과 karata=카드, mkono=패 정도만). 동아프리카 온라인 포커·베팅 씬(SportPesa 등)은 **영어 포커 용어를 그대로 사용** → 이게 자연스러운 현지화. **직역·억지 신조어 금지.** ★**직역 아님 — 현지 맥락 재구성.**

## 용어 (현지 관습 — 영어 포커 용어 유지 + 스와힐리 산문)
- **핵심 원칙**: 포커 전문 용어(족보·액션·구조)는 **영어 그대로**(blinds, small/big blind, flop, turn, river, check, bet, call, raise, fold, all-in, pot, showdown, kicker, side pot, hand rankings 등). 스와힐리 산문으로 설명·연결. 첫 등장 시 짧은 스와힐리 뜻풀이 가능.
- **족보 = 영어명 유지(첫 등장 시 스와힐리 설명)**:
  - Royal Flush → **Royal Flush** (mkono bora kabisa)
  - Straight Flush → **Straight Flush**
  - Four of a Kind → **Four of a Kind** (karata nne zinazofanana)
  - Full House → **Full House**
  - Flush → **Flush** (karata tano za suti moja)
  - Straight → **Straight** (karata tano mfululizo)
  - Three of a Kind → **Three of a Kind** (또는 trips)
  - Two Pair → **Two Pair** (jozi mbili)
  - One Pair → **Pair** (jozi moja)
  - High Card → **High Card** (karata ya juu)
  - ★홀덤 순서 = **Full House > Flush**(표준, 변형 없음).
- **액션(영어 유지 + 스와힐리 동사 병기 가능)**: check → **check**, bet → **bet / dau (kuweka dau)**, call → **call (kuita)**, raise → **raise (kupandisha)**, fold → **fold (kukunja)**, all-in → **all-in**.
- **블라인드**: Blinds → **blinds (dau la lazima)**, Small Blind → **small blind (SB)**, Big Blind → **big blind (BB)**.
- **스트리트/구조**: preflop, flop, turn, river 그대로. betting round → **raundi ya kuweka dau**, community cards → **karata za pamoja / za jamii**, hole cards → **karata za mkononi / za siri**.
- **핵심 명사(스와힐리)**: card → **karata**, hand(패) → **mkono**, hand rankings → **madaraja ya mikono / mpangilio wa mikono**, dealer → **dila / mgawaji**, button → **button (BTN)**, pot → **pot / mkusanyiko wa fedha**, main pot → **pot kuu**, side pot → **side pot / pot ya pembeni**, kicker → **kicker**, showdown → **showdown / kuonyesha karata**, suit → **suti**, stack → **stack / rundo la chipsi**, chips → **chipsi**, table → **meza**, player → **mchezaji**, money → **pesa/fedha**, Bluff → **bluff / kudanganya**, ante → **ante**.
- **포지션 약어는 영어**(첫 등장 병기): UTG, CO(Cutoff), MP, HJ, SB, BB, BTN.

## 표기 (⚠️ **영어식 계열: en·zh·ar·ms·th와 동일 — 변환 없음**)
- **숫자 = 영어식 그대로**: 천단위 `,`(1,326·198,000 그대로), 소수점 `.`(2.5·2.7:1 그대로), % `35%`. **§13 값 표기 변환 불필요**(동아프리카 영어식 관습·아라비아 숫자). 유럽식(1.326/2,5)으로 바꾸지 말 것.
- **카드 표기 라틴 유지**: `A♠`, `K♥`, `10♦`, 보드 그대로. 무늬 기호 ♠♥♦♣ 그대로.
- **화폐**: `$` 그대로(§13 보존 — KES/TZS로 바꾸지 말 것).

## 문체
- **wewe/u-**체(2인칭 단수, 개별 독자 지칭 — 스와힐리 표준). 자연스러운 스와힐리 구어체(딱딱한 사전식·기계번역 티 금지). 동아프리카 독자에게 친근하고 명료하게.
