# zh 포커 용어·표기 브리프 (번역 착수 전 웹리서치 결과)

> 대상 = **중국어 간체(zh, 简体)**. 중화권 포커 글로시(zhihu 德州扑克 전문글·legendpoker.cn·zh.wikipedia 扑克牌型·taiwanrounders 대조표) 기준. 본토(대륙) 표기 우선.

## 용어 (현지 관습)
- **행동 동사는 중국어 표준**: call → **跟注**, raise → **加注**, fold → **弃牌**, check → **过牌**, bet → **下注**, 3-bet/4-bet → 3bet/4bet 또는 三次加注(대체로 영어약어 그대로 통용).
- **영어 약어/명사 혼용 관습**: All-in → **全押**(또는 全下, 구어에선 all-in 그대로도 씀), Ante → 前注, Straddle → 那啥(드묾, straddle 그대로), Heads-up → 单挑, GTO/EV/ICM 등 약어는 영어 그대로.
- **스트리트**: Flop → **翻牌**, Turn → **转牌**, River → **河牌**, 프리플랍 → 翻牌前.
- **족보(牌型, 중요)**: 皇家同花顺(로열플러시), 同花顺(스트레이트플러시), **四条**(포카드), **葫芦**(풀하우스), 同花(플러시), **顺子**(스트레이트), **三条**(트리플/셋; set=暗三条, trips=明三条로 구분 가능), 两对(투페어), 一对(원페어), 高牌(하이카드).
- **핵심 명사**: Blinds → 盲注 (Big Blind=大盲注/大盲, Small Blind=小盲注/小盲), Showdown → **摊牌**, Kicker → **踢脚牌**(또는 单张), Pot → 底池, Main pot → 主池, Side pot → **边池**, Board/community cards → **公共牌**, Hole cards → 底牌/手牌, Pot odds → **底池赔率**, Stack → 筹码量, Buy-in → 买入, Bluff → **诈唬**(또는 唬牌).
- **포지션**: Dealer/Button(BTN) → **庄家/按钮位**, SB → 小盲位, BB → 大盲位, UTG → **枪口位**, CO(Cut Off) → **关煞位/劫杀位** = **버튼 오른쪽 첫 자리**(웹리서치로 EN 마스터 CO수정과 일치 확인), MP → 中位, HJ → 劫机位.

## 표기 (⚠️ 다른 언어와 다름)
- **숫자 = 영어와 동일**. 천단위 **콤마**(,), 소수점 **마침표**(.) → `1,326`, `198,000`, `0.003%`, `2.5 BB`, `2.7:1`. **es/pt/de식 점·콤마 반전 금지**(zh는 영어 포맷 그대로 → §13 숫자 변환 리스크 없음).
- **화폐**: $ 그대로(§13 보존).
- **문장부호**: 중국어 전각 문장부호 사용 권장(，。：；「」 또는 ""). 단 코드 템플릿 리터럴 안에서 직선 따옴표 " 는 문자열 안전(문제 없음). 제목·본문 가독성은 전각 기준.
- **간체자 철저**(繁體 혼입 금지): 关(關)·边(邊)·筹(籌)·诈(詐) 등.
