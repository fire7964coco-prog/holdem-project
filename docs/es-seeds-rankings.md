# 족보6 시드 키워드 (Spain/Spanish) — 규칙6와 동일 방식

> Generate → 전체선택 → SERP Extr(1회, 얇으면 2회차) → **Analysed only** Export → Claude에게.
> 파일명 `rankings_ES.xlsx` 식. Country=Spain, Language=Spanish 유지.
> 🎁 color/escalera/full 비교는 규칙6에서 이미 확보 → 아래는 **아직 얇은 소주제 위주**.

## ① hand-rankings (필라) — 헤드텀 보강
```
manos de poker *

escalera real *
```

## ② flush-vs-straight — (이월 많음, 보강만)
```

```

## ③ kicker — ★신규 필요
```
kicker poker *
que es el kicker en poker *
perder por kicker *
```

## ④ tiebreak (desempate) — ★신규 필요
```
desempate poker *

```

## ⑤ split-pot (bote dividido) — ★신규 필요
```

```

## ⑥ reading-the-board — ★신규 필요
```

```
