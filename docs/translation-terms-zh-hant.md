## 1. 족보 10종 (3편 만장일치 — 번체 표준)
| 순위 | 번체 | 영어 병기 |
|---|---|---|
| 1 | 皇家同花順 | Royal Flush |
| 2 | 同花順 | Straight Flush |
| 3 | 四條 | Four of a Kind |
| 4 | 葫蘆 | Full House |
| 5 | 同花 | Flush |
| 6 | 順子 | Straight |
| 7 | 三條 | Three of a Kind |
| 8 | 兩對 | Two Pair |
| 9 | 一對 | One Pair |
| 10 | 高牌 | High Card |
- ⚠️ 간체와 다름: 번체 `四條/三條`(간체 四条/三条), `兩對`(两对), `葫蘆`(葫芦).

## 2. 타이·키커·엣지 용어 (현지 실사용)
- **踢腳牌 / 踢牌** = 키커 ("若相同則比踢腳牌依序大小")
- **平手 → 分池 / 平分底池（Chop / Chop Pot）** = 스플릿 팟
- **疊牌** = 카운터핏(보드 페어로 내 페어가 최종5장에 못 들어감). 俚語 `被疊掉了`, `被4+1了`
- **同花比大小** = 플러시끼리 최고카드로 비교
- **花色不比大小** = 무늬는 서열 없음 (★`花色大小`가 인기 검색어=오개념 → FAQ로 정정)

## 3. 카드·무늬 표기
- 무늬기호+숫자: `♠7 ♠8 ♦K ♣A ♥2`. **10은 `10`**(T 금지) — `黑桃的10、J、Q、K、A`.
- 중문 무늬명도 병용 가능: **黑桃(♠) / 紅心(♥) / 方塊(♦) / 梅花(♣)**.

## 4. 문체·톤
- 초보 대상, **친근 존댓말**("務必要熟記"·"讓你看見").
- **경고 톤** 적극 사용: `千萬要認真比較`·`特別特別小心`.
- 현장 俚語 소량 양념: `被疊掉了`·`K花`·`被4+1了` (E-E-A-T 현장감).
- **팁 콜아웃**: `【貼心提醒】`·`教練提醒:` 관습.

## 5. 구조 관습 (족보글)
- H1: `完整教學`/`超詳細講解` + `初學者看這篇就夠了`.
- H2: 명사형 — `德州撲克牌型有哪些？`·`手牌大小排名一覽（從大到小）`·`三種常見的誤判情況`·`結語`.
- 족보 = **번호 리스트**(1~10). 표 사용 흔함(✅), 이모지(✅❌📌🔑💡) 관대.
- ★승부처 = **誤判情況(오판 케이스)**: `同花比大小`·`疊牌`·`平分底池` — 우리 §13 정확성으로 차별화(경쟁 娛樂城글 얇음).
