# vi 포커 용어·표기 브리프 (번역 착수 전 웹리서치 결과)

> 대상 = **베트남어(vi, Tiếng Việt)**. 베트남 포커 글로시(ggpoker.com/vi · wikipoker.net · natural8/vi · wptglobal/vi · 68gamebaivn · tinhte.vn) 기준. **베트남은 고유 포커 용어가 풍부** — 족보는 베트남어 고유명이 표준(현지인이 쓴 느낌의 핵심). 액션은 베트남어+영어 차용 혼용.

## 용어 (현지 관습)
- **족보는 베트남어 고유명 우선(첫 등장 시 영어 병기)**:
  - Royal Flush → **Sảnh Thượng** (또는 Thùng Phá Sảnh Chúa)
  - Straight Flush → **Thùng Phá Sảnh**
  - Four of a Kind → **Tứ Quý**
  - Full House → **Cù Lũ**
  - Flush → **Thùng**
  - Straight → **Sảnh**
  - Three of a Kind(set) → **Xám / Bộ Ba** (set = xám chi)
  - Two Pair → **Hai Đôi** (또는 Thú)
  - One Pair → **Một Đôi / Đôi**
  - High Card → **Mậu Thầu / Bài Cao**
- **행동(베트남어+영어 혼용, 자연스러운 쪽)**: check → **check**(그대로; ⚠️ "kiểm tra"는 오역, 쓰지 말 것), bet → **cược/đặt cược**, call → **theo (call)**, raise → **tố (raise)**, fold → **bỏ bài (fold)**, all-in → **all-in**(또는 tố hết/theo hết).
- **블라인드**: Blinds → **mù** (Small Blind → **mù nhỏ (SB)**, Big Blind → **mù lớn (BB)**). 영어 SB/BB 병기 가능.
- **스트리트/구조**: Flop, Turn, River, pre-flop 그대로. betting round → **vòng cược**, round → **vòng**. ⚠️ **Sảnh(스트레이트 족보) ≠ vòng(스트리트)** 혼동 금지.
- **핵심 명사**: card → **bài/lá bài**, community cards → **bài chung**, hole cards → **bài tẩy** (또는 hole cards), hand(패) → **tay bài**, hand rankings → **thứ hạng bài / xếp hạng bài**, dealer → **người chia bài / Dealer**, button → **nút Dealer (BTN)**, pot → **pot / hũ (tiền)**, main pot → **pot chính**, side pot → **pot phụ / side pot**, kicker → **kicker / lá phụ**, showdown → **lật bài (showdown)**, suit → **chất**, stack → **stack / chồng chip**, Bluff → **tố lừa / bluff**.
- **포지션 약어는 영어**(첫 등장 병기): UTG, CO(Cutoff)=버튼 오른쪽 첫 자리, MP, HJ, SB, BB, BTN.

## 표기 (⚠️ 중요 — es/pt/de/id와 동일 계열)
- **숫자 = 베트남식**: 천단위 **마침표(.)**, 소수점 **쉼표(,)** → `1.326`, `198.000`, `0,003%`, `2,5 BB`, `2,7:1`. **영어식 1,326 / 2.5 → 반드시 변환.** (§13 숫자값 보존, 구분자만 베트남식)
- **성조 부호(dấu) 철저**: 모든 베트남어 단어는 올바른 성조/모음 부호 표기(Sảnh·Thùng·Cù Lũ·tố·bỏ·mù 등). 부호 누락·오타 = 신뢰 리스크(§13급).
- **카드 표기 라틴 유지**: `A♠`, `K♥`, `10♦`, 보드 그대로. 무늬 기호 ♠♥♦♣ 그대로.
- **화폐**: $ 그대로(§13 보존).

## 문체
- **bạn**체(2인칭, 베트남 콘텐츠 표준 — 친근·중립). 자연스러운 베트남 구어체(딱딱한 직역 금지). 포커 커뮤니티가 실제 쓰는 표현.
