# zh 포커 용어·표기 브리프 (번역 착수 전 웹리서치 결과)

> 대상 = **중국어 간체(zh-CN, 简体, 대륙 우선)**. 홍콩/대만 아님(대만은 「德州撲克」·「梭哈」 등 다름 → 참고만).
> 목표: 直译 금지 — 중국 德扑 커뮤니티(公众号·知乎·传奇/摩十客·中扑网)에서 실제 쓰는 어투로 재저작. 정확성(§13)은 언어 불변.
> (2026-07 웹리서치로 5개+ 중문 출처 정독 후 확장. 하단 出处 참조.)

---

## 0. 핵심 결론 (먼저 읽기)

1. **중국 牌手 = 中英 혼용.** 기본 동작/구조어는 중국어(跟注·加注·翻牌), 그러나 고빈도 일부는 **영어 그대로**: `all in`·`3bet`·`c-bet`·`GTO`·`ICM`·`range`·`set`·`bluff`·`bad beat`·`cooler`. 교학문은 **中文(English)** 첫 등장 시 괄호 병기가 가장 안전.
2. **족보는 전부 중국어**, 대륙 표준형 유일(皇家同花顺·四条·葫芦…). §2.
3. **어투는 「你」** — 구어·직접·公众号/知乎 교학체. 「您」 지양.
4. **숫자는 아라비아**, 大额은 「万/亿」(3万·198万), 비율은 **3:1**(반각 콜론), 화폐 `$` 보존(§13).
5. **스택 깊이는 BB 단위**(100BB·20BB), 벳 사이즈는 3BB / 半池 / 1/2 底池.

## 1. 保留英文 vs 译成中文 (逐词)

> 「首选」= 본문 기본 표기 · 「英文并存」= 중국 테이블에서 영어도 그대로 쓰는지 · 「动词化」= 구어 동사.

### A. 구조/스트리트 (→ 전부 중국어)

| 英文 | 首选中文 | 英文并存 | 备注 |
|---|---|---|---|
| blinds | 盲注 | 少 | 통칭 |
| small blind (SB) | 小盲 / 小盲注 | SB | 포지션은 「小盲位」 |
| big blind (BB) | 大盲 / 大盲注 | BB(고빈도) | BB = 「단위」(§3) 兼 「大盲位」 |
| flop | 翻牌 | 少 | 翻牌圈 |
| turn | 转牌 | 少 | 转牌圈 |
| river | 河牌 | 少 | 河牌圈. **가짜친구**: ≠ 河流 |
| preflop | 翻前 / 翻牌前 | preflop | 「翻前」이 가장 구어 |
| board | 公共牌 / 牌面 | board | 牌面=그 5장의 형태, 公共牌=범칭. **가짜친구**: ≠ 木板 |
| showdown | 摊牌 / 亮牌 | 少 | 「摊牌」 |
| hole cards | 底牌 / 手牌 | — | |

|---|---|---|---|
| fold | 弃牌 / 盖牌 | **fold 구어 많이 씀** | "我 fold 了" / "直接弃" / "盖牌" |

| straddle | 抓("抓一个") / 加码盲注 | **straddle 그대로** | "谁抓?" / "straddle" |

### D. 三条 하위분류 (set/trips 반드시 구분 — §5도 참조)

| 英文 | 首选中文 | 备注 |
|---|---|---|
| set | 暗三条 | 포켓페어 + 공공1장. 구어 "我 set 了" |
| trips | 明三条 | 핸드1장 + 공공2장. 구어 trips |

> **주의**: set/trips 둘 다 중국어로는 「三条」지만 은닉성·강도가 다름. 牌力/은닉성 비교 맥락에선 **반드시 「暗三条(set)/明三条(trips)」로 번역+영어 병기**.

## 2. 족보(Hand Rankings) 표준 중국어 — 대륙 唯一形

> 交叉验证 일치(维基 撲克牌型 + 传奇扑克 + dpskill + moshike). 큰 것부터:

| 英文 | **简体标准** | 备选/구어 | 定义 |
|---|---|---|---|
| Royal Flush | **皇家同花顺** | 同花大顺 | 同花 10-J-Q-K-A |
| Straight Flush | **同花顺** | — | 同花 5연속 |
| Four of a Kind | **四条** | 铁支(粤/구) | 同点 4장. 대륙 首选=**四条** |
| Full House | **葫芦** | 满堂红(비유일 뿐, 정식 아님) | 三带二. **首选 葫芦** |
| Flush | **同花** | — | 同花 5장 非연속 |
| Straight | **顺子** | 顺 | 5연속 非同花. **가짜친구**: ≠ 直的 |
| Three of a Kind | **三条** | 三张(구) | 하위 set/trips §1D |
| Two Pair | **两对** | ⚠️两队(오타 흔함, 금지) | 두 페어 |
| One Pair | **一对** | 对子 | |
| High Card | **高牌** | 散牌 | 무성패, 최대 单张 비교 |

**译者 통일 규칙**:
- Full House → 무조건 **葫芦**(「满堂红」은 비유로만).
- Four of a Kind → 무조건 **四条**(「铁支」= 粤語/구칭, 금지).
- Two Pair → **两对**(「两队」로 잘못 쓰지 말 것).
- 轮子 = A-2-3-4-5 = 최소 顺子 (A가 1).

## 3. 숫자 / 화폐 / 포맷 (⚠️ es/pt/de와 다름)

- **아라비아 숫자** 전부. 수량에 한자수(一二三) 쓰지 말 것.
- **숫자 포맷 = 영어와 동일**: 천단위 **콤마**, 소수점 **마침표** → `1,326` · `0.003%` · `2.5BB` · `2.7:1`. **점·콤마 반전 금지**(§13 변환 리스크 없음).
- **大额 단위**: 「万 / 亿」이 地道. 예 `3万筹码` · `盲注 5000/1万` · `奖池 198万` · `100万码量, 大盲10万 → 10BB`. (EN 1,000,000 → 中 100万.)
- **비율**: **`3:1`**(반각 콜론) = SEO/본문 首选. 문장 속 구어는 「3比1」도 가능. 제목은 `3:1`.
- **백분율**: `33%` · `约33.3%`.
- **화폐**: §13 대로 `$` 保存(`$2`·`$50`). 대륙 원어민은 「刀/美元」도 쓰나 프로젝트 일관성상 `$` 유지, `¥`로 바꾸지 말 것.
- **BB 표기**: `3BB`·`100BB`·`20BB`(숫자 붙여쓰기). 벳 사이즈 `半池`·`1/2 底池`·`1/3 底池`·`2/3 底池`.
- **카드 표기**: EN의 `A♠ K♦` 유지 또는 「黑桃A 方块K」. `AKs`(同花/suited)·`AKo`(非同花/offsuit) 圈内 통용 → 保留 가능.

## 6. 문체 / 称呼

- **「你」체.** 德扑圈 公众号·知乎·传奇/摩十客 표준. 구어·직접·圈内 친구감. 「您」 금지(과격식).
- 어투 예: 「翻前拿到 AK 你到底该 3bet 还是跟?」 「别急着 all in, 先看底池赔率.」 → **짧은 문장·동사 선두·행동 직호**.
- 적당한 속어로 地道감: 「锅」(底池)·「河了他」(river 역전)·「背」(운 나쁨). 단 교학 본문은 절제, 흑화는 양념만.
- 기계번역 냄새 제거: 「一个玩家(a player)」 축자직역 금지(→「有人/对手」); 「进行下注/进行加注」 남용 금지(→ 「下注/加注」).
- **간체자 철저**(繁體 혼입 금지): 关(關)·边(邊)·筹(籌)·诈(詐) 등.
- 문장부호는 전각(，。：；「」"") 권장. 단 코드 템플릿 리터럴 안 직선 " 는 안전.

# 8. 클러스터별 SERP winnable 리서치 (경화·신규 포스팅 재사용)
