# ro 포커 용어·표기 브리프 (번역 착수 전 웹리서치 결과)

> 대상 = **루마니아어(ro, Română)**. 루마니아 포커 글로시(888poker.ro · vladcazino.ro · unibet.ro · ggpoker.com/ro · ro.wikipedia Lista_de_mâini_de_poker · supercazino.ro) 기준. 루마니아는 **규제 포커 시장**·고유 포커 용어 확립. ★**직역 아님 — 현지 맥락 재구성.**

## ⚠️ 치명 함정 (§13급)
1. **Culoare = Flush** (프랑스 couleur·이탈리아 colore·폴란드 kolor처럼). ⚠️색/무늬(suit)도 culoare → 문맥 구분. flush 뜻일 때 족보 Culoare.
2. **Chintă = Straight**, **Chintă de Culoare = Straight Flush**, **Chintă Roială = Royal Flush**. Chintă 단독 = straight.
3. **Careu = Four of a Kind**(포카드).
4. **홀덤 순서 = Full House > Culoare**(풀하우스 > 플러시, 루마니아 공식 clasament도 동일 — 표준). 변형 없음.

## 용어 (현지 관습)
- **족보 (루마니아어 표준명, 첫 등장 시 영어 병기)**:
  - Royal Flush → **Chintă Roială**
  - Straight Flush → **Chintă de Culoare**
  - Four of a Kind → **Careu**
  - Full House → **Full House** (또는 Full)
  - Flush → **Culoare** ⚠️(flush)
  - Straight → **Chintă**
  - Three of a Kind / set → **Trei de un fel** (set)
  - Two Pair → **Dublă Pereche** (또는 Două perechi)
  - One Pair → **Pereche**
  - High Card → **Carte Mare**
- **액션**: check → **check / a checka**, bet → **pariu / a paria**, call → **a plăti / a vedea (call)**, raise → **a mări / raise**, fold → **a renunța / fold (a se retrage)**, all-in → **all-in / tot potul**. 첫 등장 후 자연스러운 쪽.
- **블라인드**: Blinds → **blind-uri**, Small Blind → **small blind (SB)**, Big Blind → **big blind (BB)**. 영어 SB/BB 병기 흔함.
- **스트리트/구조**: preflop, flop, turn, river 그대로(루마니아도 영어 유지). rundă de pariere(betting round), cărți comune(community cards), cărți din mână/închise(hole cards).
- **핵심 명사**: card → **carte** (pl. cărți), hand(패) → **mână**, hand rankings → **clasamentul mâinilor / ierarhia mâinilor**, dealer → **dealer / împărțitor**, button → **buton (BTN)**, pot → **pot**, main pot → **pot principal**, side pot → **side pot / pot secundar**, kicker → **kicker**, showdown → **showdown / arătarea cărților**, suit → **culoare (simbol)**, stack → **stack / teanc**, chips → **jetoane**, Bluff → **cacealma / bluff**, ante → **ante**.
- **포지션 약어는 영어**(첫 등장 병기): UTG, CO(Cutoff), MP, HJ, SB, BB, BTN.

## 표기 (⚠️ 유럽식 계열 — es·pt·de·it와 동일)
- **숫자 = 루마니아식(유럽식)**: 천단위 **마침표(.)** → `1,326`→**`1.326`**, `198,000`→**`198.000`**, `1,300`→**`1.300`**. 소수점 **쉼표(,)** → `2.5`→**`2,5`**, `2.7:1`→**`2,7:1`**, `0.84`→**`0,84`**.
- **퍼센트**: `35%` 붙여쓰기, 소수는 쉼표 `0,84%`.
- **§13 값 자체는 불변, 구분자만 루마니아식.** 라틴 숫자(0-9).
- **카드 표기 라틴 유지**: `A♠`, `K♥`, `10♦`, 보드 그대로. 무늬 기호 ♠♥♦♣ 그대로(족보 Culoare와 무늬는 문맥 구분).
- **화폐**: `$` 그대로(§13 보존 — RON/lei로 바꾸지 말 것).
