# 필라-클러스터 로드맵 & 발행·번역 워크플로우

> 이 사이트의 콘텐츠 구조 + 내부링크 + 발행/번역 순서.
> 글을 발행할 때마다 계속 업데이트하는 살아있는 문서.
> 함께 보는 문서: `PROJECT.md`, `SEO-COPYWRITING-GUIDE.md`.

---

## 1. 콘텐츠 아키텍처 (필라 → 클러스터)

- **필라(Pillar)**: 큰 주제 허브 글. 폭넓은 키워드, 클러스터로 내부링크.
- **클러스터(Cluster)**: 롱테일·세부 질문 글. 필라로 역링크.
- **전략**: 경쟁 약한 **클러스터부터** 색인을 쌓고 → 필라로 묶어 토픽 권위를 형성.

```text
[필라] 홀덤 전략 기초
 ├─ 포지션이란
 ├─ 핸드 레인지란
 └─ 베팅 라운드 흐름

[필라] 프리플랍 전략
 ├─ 버튼 포지션이 유리한 이유
 ├─ 3벳 전략
 ├─ 오픈 레인지 차트
 └─ 블라인드 디펜스

[필라] 포스트플랍 전략
 ├─ C-벳 기준
 ├─ 팟 오즈 계산
 ├─ 아웃츠 & 4-2 룰
 └─ 보드 텍스처 읽기

[필라] 쇼다운 & 룰
 ├─ 족보 순위
 ├─ 타이브레이크/키커
 └─ 스플릿 팟

[필라] 토너먼트 전략
 ├─ ICM 기초
 ├─ 숏스택 운영
 └─ 버블 플레이
```

> ⚠️ **모든 글은 이 사이트에서 처음부터 새로 쓴 오리지널.** 위 항목은 "다룰 주제"일 뿐, 외부 글을 복사·번역해 옮기지 않는다(중복 콘텐츠 = 검색 점수 하락).

## 2. 우선순위 로드맵

### Phase 1 — 클러스터 시딩 (경쟁 약한 롱테일)
- 버튼 포지션 / 3벳 / 팟 오즈 / 타이브레이크 / 4-2 룰 / 오픈 레인지
- 한국어 검수 발행 → 영어 마스터 → 색인 확인.

### Phase 2 — 필라 구축 & 내부링크
- 5개 필라 허브 작성 → 클러스터들을 묶어 토픽 권위 형성.
- 우선 언어(es, pt-BR, de, ru) 번역 시작.

### Phase 3 — 전 언어 확장
- 필라+클러스터 세트를 18개 언어로 확장.
- 토너먼트/ICM 등 심화 클러스터 추가.

---

## 3. 내부링크 규칙

- 모든 클러스터 → 소속 필라로 1회 이상 링크.
- 필라 → 주요 클러스터 전부 링크 (허브 역할).
- 클러스터 ↔ 클러스터: 문맥상 관련될 때만 (예: 버튼 포지션 ↔ 3벳, 팟 오즈 ↔ 4-2 룰).
- 링크는 **같은 언어 내에서만** (`/es/...`는 `/es/...`끼리).
- 앵커 텍스트에 대상 글의 핵심 키워드 포함.

## 4. 발행 · 번역 워크플로우 ⭐

```text
┌─────────────────────────────────────────────────────────┐
│ 1. 한국어(ko) 초안 작성  ← 운영자가 직접 작성             │
│    └ 경쟁 분석(SEO 가이드) → 작성 → 직접 검수             │
│            ↓ 검수 통과                                    │
│ 2. 영어(en) 마스터 생성  ← 한국어 기반 번역·현지화        │
│    └ 단순 직역 X, 영어 SERP 톤으로 현지화 = 번역 기준 원본│
│            ↓ 확정                                         │
│ 3. 영어 마스터 기반 16개 언어 번역 발행                   │
│    └ es, pt-BR, de, ru → 점진 확장 → 전 언어             │
└─────────────────────────────────────────────────────────┘
```

**핵심:** 한국어는 운영자가 직접 검수하는 **품질 게이트**, 영어는 모든 번역의 **단일 기준(마스터)**.

### 단계별 체크
| 단계 | 입력 | 작업 | 출력 | 상태 라벨 |
|---|---|---|---|---|
| 1 | 키워드 | 경쟁분석 → ko 초안 → 검수 | ko 본문 | Draft (ko) |
| 2 | ko 확정본 | en 현지화 번역 | en 마스터 | Master (en) |
| 3 | en 마스터 | 언어별 번역 발행 | 각 언어 페이지 | Translated |

### 워크플로우 주의
- **2단계(en)는 직역 금지** — 영어가 마스터이자 번역 품질의 천장. 어색하면 전 언어로 퍼진다.
- 한국어 원문 수정 시 → en 마스터 갱신 → 영향 언어 재번역 (`Needs update` 표시).
- 슬러그(영어)는 모든 언어 공유, 절대 안 바꾼다.
