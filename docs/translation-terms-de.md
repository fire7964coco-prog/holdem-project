# de 포커 용어·표기 브리프 (번역 착수 전 웹리서치 결과)

> 대상 = **독일어(de)**. 독일 포커 글로시(poker.de·pokerzeit·pokerolymp·cardplayer.de·de.wikipedia) 기준.

## 용어 (현지 관습)
- **Denglisch 동사(표준)**: call → **callen**(또는 mitgehen/bezahlen), raise → **raisen**(또는 erhöhen), fold → **folden**(또는 passen/aussteigen), check → **checken**(또는 schieben), 3-bet/4-bet 그대로.
- **영어 그대로(대문자 명사)**: Blinds(Small Blind/Big Blind), All-in, Showdown, Side Pot(=Nebenpot도 가능), Kicker, Board, Flop/Turn/River, Pot, Dealer-Button/Button, Stack, Buy-in, Ante, Straddle, Heads-up, Pot Odds, Bluff.
- **족보는 독일어(중요)**: Höchste Karte(하이카드), Paar(원페어), Zwei Paare(투페어), **Drilling**(트리플/셋), **Straße**(스트레이트), Flush, Full House, **Vierling**(포카드/쿼드), Straight Flush, Royal Flush.
- main pot → Hauptpot, side pot → Nebenpot/Side Pot.

## 표기
- **명사 대문자화** 철저(Blinds, Pot, Einsatz, Karten, Runde …).
- **숫자**: 천단위 **점**(.), 소수점 **콤마**(,) → `1.326`, `198.000`, `0,003 %`(독일은 % 앞 공백 관습, 없어도 무방), `2,5 BB`, `2,7 zu 1`.
- **화폐**: 독일은 €지만 온라인 포커는 $ 일반 → §13 보존 위해 **$ 그대로**.
