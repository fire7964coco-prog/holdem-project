# fr 포커 용어·표기 브리프 (번역 착수 전 웹리서치 결과)

> 대상 = **프랑스어(fr, Français)**. 프랑스 포커 글로시(pokerstars.fr/learn · fr.pokernews.com · ggpoker.com/fr · fr.pokerlistings.com · fr.wikipedia Main_au_poker · pokerfrance.biz) 기준. 프랑스는 **고유 포커 용어가 확립** — 족보·액션 모두 프랑스어 표준명이 있고 일부 영어 병용. ★**직역 아님 — 현지 맥락 재구성**(translation-spec "★★ 이 작업의 본질" 참조). 대상 시장 = 프랑스·벨기에·스위스·퀘벡·아프리카 프랑코폰.

## 용어 (현지 관습)
- **족보 = 프랑스어 표준명 우선(첫 등장 시 영어 병기 가능)**:
  - Royal Flush → **Quinte Flush Royale**
  - Straight Flush → **Quinte Flush**
  - Four of a Kind → **Carré**
  - Full House → **Full** (full aux X par les Y)
  - Flush → **Couleur** ⚠️(=flush. 프랑스 카드에서 "무늬"도 couleur → 문맥 구분 필수. flush 뜻일 땐 족보, 무늬 뜻일 땐 "enseigne/couleur")
  - Straight → **Quinte** (또는 Suite)
  - Three of a Kind / set → **Brelan** (set = brelan servi)
  - Two Pair → **Double Paire** (또는 Deux Paires)
  - One Pair → **Paire**
  - High Card → **Carte Haute** (또는 Hauteur)
- **액션(프랑스어 표준 + 영어 병용)**: check → **checker / parole** ("parole"=체크 프랑스어), bet → **miser / mise**, call → **suivre** (caller도 통용), raise → **relancer / relance**, fold → **se coucher** (folder/jeter도), all-in → **tapis (faire tapis)** / all-in. 첫 등장 후 자연스러운 쪽.
- **블라인드**: Blinds → **blindes**, Small Blind → **petite blinde (SB)**, Big Blind → **grosse blinde (BB)**. 영어 SB/BB 병기 가능.
- **스트리트/구조**: pré-flop, flop, turn(**tournant**), river(**rivière**) — flop/turn/river 영어도 통용, 프랑스어 tournant/rivière 자연스러움. tour d'enchères(betting round), cartes communes(community cards), cartes fermées(hole cards).
- **핵심 명사**: card → **carte**, hand(패) → **main**, hand rankings → **classement des mains**, dealer → **donneur**, button → **bouton (BTN)**, pot → **pot**, main pot → **pot principal**, side pot → **pot annexe / side pot**, kicker → **kicker**, showdown → **abattage / showdown**, stack → **tapis / stack**(⚠️ tapis=all-in과 stack 둘 다 → 문맥), chips → **jetons**, Bluff → **bluff**, ante → **ante**.
- **포지션 약어는 영어**(첫 등장 병기): UTG, CO(Cutoff), MP, HJ, SB, BB, BTN.

## 표기 (⚠️ 최대 함정 — 프랑스식은 **de·es와 또 다름!**)
- **숫자 = 프랑스식**: 천단위 **공백**(espace) → `1,326`→**`1 326`**, `198,000`→**`198 000`**, `1,300`→**`1 300`**. ⚠️**마침표(.) 쓰지 말 것**(그건 독일/스페인식). 소수점 **쉼표(,)** → `2.5`→**`2,5`**, `2.7:1`→**`2,7:1`**, `0.84`→**`0,84`**.
- **퍼센트 앞 공백**(프랑스 조판 관습): `35%`→**`35 %`**, `0.84%`→**`0,84 %`**, `8%`→**`8 %`**.
- **§13 값 자체는 불변, 구분자·공백만 프랑스식.** 라틴 숫자(0-9).
- **카드 표기 라틴 유지**: `A♠`, `K♥`, `10♦`, 보드 그대로. 무늬 기호 ♠♥♦♣ 그대로(족보 "couleur"와 무늬는 문맥 구분).
- **화폐**: `$` 그대로(§13 보존 — €로 바꾸지 말 것).

## 문체
- **tu**체(친근 2인칭). 사이트의 타 언어 친근체(es=tú·de=du·tr=sen·vi=bạn)와 일관. 자연스러운 프랑스 포커 구어(딱딱한 vous 격식체·기계번역 티 금지). 프랑스 포커 커뮤니티(클럽·온라인)가 실제 쓰는 표현·리듬.
- 프랑스어 문법·성수 일치·아상(accents é/è/ê/à/ç/ù) 정확. 철자·악상 오타 = §13급 신뢰 리스크.
