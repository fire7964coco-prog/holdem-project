# ru 포커 용어·표기 브리프 (번역 착수 전 웹리서치 결과)

> 대상 = **러시아어(ru, Русский)**. 러시아 포커 글로시(poker.ru · pokeroff.ru · funfarm.pro · gipsyteam.ru · championat.com/poker · worldpokerdeals.com/ru · pokerhouse.ru) 기준. 러시아는 **고유+음차 포커 용어 확립**. ★**직역 아님 — 현지 맥락 재구성**. 대상 = 러시아+CIS(우크라이나·카자흐스탄·벨라루스 등 러시아어권).

## 용어 (현지 관습)
- **족보 (러시아어 표준명 — 대부분 영어 음차, 첫 등장 시 영어 병기)**:
  - Royal Flush → **Роял-флеш** (또는 Флеш-рояль)
  - Straight Flush → **Стрит-флеш**
  - Four of a Kind → **Каре**
  - Full House → **Фулл-хаус**
  - Flush → **Флеш**
  - Straight → **Стрит**
  - Three of a Kind / set → **Сет** (또는 Тройка)
  - Two Pair → **Две пары**
  - One Pair → **Пара**
  - High Card → **Старшая карта**
- **액션(러시아어 음차+동사 병용)**: check → **чек** (또는 чекнуть), bet → **ставка / бет**, call → **колл** (уравнять/заколлировать), raise → **рейз** (повысить/поднять), fold → **фолд** (сбросить/спасовать), all-in → **олл-ин** (또는 ва-банк). 첫 등장 후 자연스러운 쪽.
- **블라인드**: Blinds → **блайнды**, Small Blind → **малый блайнд (SB)**, Big Blind → **большой блайнд (BB)**. 영어 SB/BB 병기 가능.
- **스트리트/구조**: префлоп, флоп, терн, ривер (러시아 표준 음차). раунд торговли(betting round), борд(board), общие карты(community cards), карманные карты(hole cards).
- **핵심 명사**: card → **карта**, hand(패) → **рука** (또는 комбинация), hand rankings → **комбинации / старшинство рук**, dealer → **дилер**, button → **баттон / кнопка (BTN)**, pot → **банк** (또는 пот), main pot → **основной банк**, side pot → **сайд-пот / побочный банк**, kicker → **кикер**, showdown → **вскрытие / шоудаун**, suit → **масть**, stack → **стек**, chips → **фишки**, Bluff → **блеф**, ante → **анте**.
- **포지션 약어는 영어**(첫 등장 병기): UTG, CO(Cutoff), MP, HJ, SB, BB, BTN.

## 표기 (⚠️ 유럽식 계열 — 프랑스와 유사)
- **숫자 = 러시아식(GOST)**: 천단위 **공백**(пробел) → `1,326`→**`1 326`**, `198,000`→**`198 000`**, `1,300`→**`1 300`**. ⚠️마침표·콤마 천단위 금지. 소수점 **쉼표(,)** → `2.5`→**`2,5`**, `2.7:1`→**`2,7:1`**, `0.84`→**`0,84`**.
- **퍼센트**: 러시아 포커 웹 관습상 `35%`(붙여쓰기) 또는 `35 %` 둘 다 통용 → **`35%` 붙여쓰기 권장**(일관되게). 소수 %는 쉼표: `0.84%`→`0,84%`.
- **§13 값 자체는 불변, 구분자·공백만 러시아식.** 라틴 숫자(0-9) — 카드/수치는 아라비아 숫자 그대로.
- **카드 표기 라틴 유지**: `A♠`, `K♥`, `10♦`, 보드 그대로. 무늬 기호 ♠♥♦♣ 그대로.
- **화폐**: `$` 그대로(§13 보존 — ₽/руб로 바꾸지 말 것).

## 문체
- **ты**체(친근 2인칭). 사이트 타 언어 친근체(tu·du·sen·tú)와 일관. 러시아 포커 커뮤니티·스트리머·가이드가 실제 쓰는 자연스러운 구어(딱딱한 관공서식·기계번역 티 금지). 러시아 포커 씬 표현·리듬.
- 키릴 문자 철자·격변화(падежи)·동사상(вид) 정확. 철자 오타 = §13급 신뢰 리스크.
