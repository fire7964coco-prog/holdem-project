# 홀덤펍 소개 — 유료 티어 기획서 (미래 실행용)

> 작성 2026-07-11. 관련 발행글: `/blog/holdem-pub-promotion`(무료 소개 신청 안내).
> **지금은 실행하지 않는다.** 무료 등재로 매장·유입·신뢰를 먼저 쌓고, 아래 트리거가 충족되면 유료 티어를 얹는다.
> 발행글에 이미 *"기본 무료 + 유료 제휴 콘텐츠엔 '광고' 표기"* 를 깔아둬서, 이 문서대로 티어만 추가하면 됨(기존 글 구조 변경 불필요).

---

## 0. 원칙 (절대 안 바꿈)

1. **무료 기본 등재는 영구 유지.** 유료는 그 위에 얹는 '노출 강화·전용 콘텐츠' 상품일 뿐. → 에디토리얼 신뢰(=SEO·E-E-A-T 자산) 보호.
2. **"순위 판매" 프레이밍 금지.** "돈 내면 1등"이 아니라 "노출/콘텐츠 상품".
3. **유료일수록 합법성 검증 더 빡세게.** 돈 받고 소개한 곳이 불법(환전)이면 사이트 책임이 더 무겁다.
4. **모든 유료 콘텐츠에 '광고' 표기** (공정위 「추천·보증 등에 관한 표시·광고 심사지침」).

## 1. 유료 도입 트리거 (언제 시작하나)

아래를 **대체로 충족**하면 유료 티어 오픈:

- [ ] 무료 등재 매장 **약 20~30곳 이상** 확보(디렉토리로서 최소 규모)
- [ ] "홀덤마스터 소개 → 매장 유입/클릭" **증거 데이터** 확보(GA/GSC로 지역 페이지·소개글 유입 추적)
- [ ] 지역별 소개 페이지 or 디렉토리 구조가 갖춰짐(무료 등재가 실제로 노출되는 자리)
- [ ] 접수 채널 안정화(구글 폼/도메인 이메일)

> 이유: 펍이 돈 내려면 "여기 소개하면 손님 온다"는 실적이 먼저 있어야 한다(닭-달걀). 실적 없이 과금 = 스팸 디렉토리와 동급.

## 2. 티어 구조 (제안)

| 티어 | 내용 | 노출 | 광고표기 |
|---|---|---|---|
| **무료 기본 등재** | 상호·주소·영업시간·정기대회·사진 1~2장, 검수 후 지역 목록 등재 | 지역 목록 일반 노출 | 불필요(에디토리얼) |
| **① 전용 소개글**(스폰서드) | 매장 단독 소개 콘텐츠 1편(상세·사진 다수·인터뷰형) | 블로그 글 1편 + 지역 페이지 링크 | **필수** |
| **② 상단·우선 노출** | 지역 목록 상단 고정 또는 강조 뱃지 | 지역 페이지 상단 | **필수**('추천/광고' 뱃지) |
| **③ 배너 노출** | 관련 글·지역 페이지 배너 | 지정 위치 배너 | **필수** |
| **(선택) 대회·이벤트 홍보** | 정기 토너/이벤트 우선 노출 | 대회 섹션 | **필수** |

- **번들**(①+②) 할인 제공 가능.
- 무료 등재 매장이 유료로 업그레이드하는 구조(무료가 유입 깔때기).

## 3. 가격 (제안 — ★반드시 시장 검증 후 확정)

> 아래는 국내 지역업소 광고/체험단 관행 기준의 **가설**이다. 실제 확정 전에 경쟁사(홀덤펍몬·포커라이프 광고 단가 등)와 실제 문의 반응으로 검증할 것.

| 상품 | 제안 가격(가설) | 과금 |
|---|---|---|
| ① 전용 소개글 | 1편 **5~15만원**(1회성) 또는 월 3~5만원 유지 | 1회성/월 |
| ② 상단·우선 노출 | 월 **3~10만원** | 월 구독 |
| ③ 배너 | 위치별 월 **5~20만원** | 월 구독 |
| 번들(①+②) | 개별 합 대비 **10~20% 할인** | — |

- **선불** 원칙. 게시 중단(불법 확인 등) 시 환불 정책 명문화(4항 계약 조건).
- 초기엔 **런칭 할인/무료 체험 1개월**로 첫 유료 고객 확보.

## 4. 법적·운영 가드레일 (계약 조건에 포함)

- **자격**: 사업자등록 + 일반음식점 영업신고 완료, 사행성(환전·참가비 시상) 없음. 유료는 사업자등록증 사본 필수 확인.
- **게시 중단 조항**: "칩 환전·참가비 시상 등 사행성 운영이 확인되거나 관계기관 단속·행정처분이 있으면 **환불 없이 즉시 게시 중단**." (돈 받았어도 불법이면 즉시 내림)
- **광고 표기**: 유료 콘텐츠 제목/본문 최상단에 `광고`. 배너·상단노출엔 '추천(광고)' 뱃지.
- **책임 한계**: 게재는 적법성 인증·보증이 아님(기존 발행글 고지 그대로 승계).
- **온라인 홀덤/총판/머니상**: 유료여도 **일절 불가**(기존 배제 원칙 유지). ← 미래 온라인(해외 라이선스) 사업은 **별도 도메인·별도 법인**으로 분리하고 이 clean 사이트와 섞지 않는다.

## 5. 광고 표기 문구 (실제 사용 카피)

- **전용 소개글 상단**:
  > `광고` · 본 콘텐츠는 [매장명]의 유료 제휴로 제작되었습니다. 홀덤마스터는 합법 운영 요건을 확인해 게재하며, 게재가 해당 매장의 적법성을 보증하지는 않습니다.
- **상단 노출 뱃지**: `추천(광고)`
- **배너 하단**: `광고 · [매장명]`
- **디렉토리 안내(무료/유료 구분 고지)**:
  > 홀덤마스터의 매장 소개는 기본 무료입니다. '광고'로 표기된 항목만 유료 제휴 콘텐츠입니다.
