# th 포커 용어·표기 브리프 (번역 착수 전 웹리서치 결과)

> 대상 = **태국어(th, ภาษาไทย)**. 태국 포커 글로시(ggpoker.com/th · thaipokerbeast.com · natural8.com/th · th.wikipedia · dummy.gameindy.com) 기준. 태국은 **영어 음차 우세 + 일부 고유 용어**. ★**직역 아님 — 현지 맥락 재구성**.

## ⚠️ 태국어 표기 특성 (중요)
- **태국어는 단어 사이 띄어쓰기가 없다**(공백은 절/문장 구분에만). 자연스러운 태국어로 붙여 쓸 것. 라틴 영어 용어(Texas Hold'em, all-in 등) 삽입 시 앞뒤 공백 자연스럽게.
- **숫자 = 아라비아 숫자(0-9)**. 태국 숫자(๐-๙) 쓰지 말 것(현대 디지털/게이밍 표준이 아라비아).

## 용어 (현지 관습)
- **족보 (태국어 음차+고유명, 첫 등장 시 영어 병기)**:
  - Royal Flush → **รอยัลฟลัช**
  - Straight Flush → **สเตรทฟลัช**
  - Four of a Kind → **โฟร์การ์ด** (또는 โฟร์ออฟอะไคนด์)
  - Full House → **ฟูลเฮาส์** (= 1 ตอง + 1 คู่)
  - Flush → **ฟลัช**
  - Straight → **สเตรท**
  - Three of a Kind / set → **ตอง** (ตอง = 고유 태국어 trips)
  - Two Pair → **สองคู่**
  - One Pair → **คู่** (또는 วันแพร์)
  - High Card → **ไพ่สูง** (또는 ไฮการ์ด)
  - ★홀덤 순서 = **ฟูลเฮาส์ > ฟลัช**(풀하우스>플러시 표준, 태국 소스도 동일 — 변형 없음).
- **액션(태국어+영어 음차)**: check → **เช็ค**, bet → **เดิมพัน / เบท**, call → **เรียก / คอล**, raise → **เพิ่มเดิมพัน / เรซ**, fold → **หมอบ / พับ / โฟลด์**, all-in → **ออลอิน**. 첫 등장 후 자연스러운 쪽.
- **블라인드**: Blinds → **บลายด์**, Small Blind → **สมอลบลายด์ (SB)**, Big Blind → **บิ๊กบลายด์ (BB)**. 영어 SB/BB 병기 가능.
- **스트리트/구조**: พรีฟลอป(preflop), ฟลอป(flop), เทิร์น(turn), ริเวอร์(river). รอบเดิมพัน(betting round), ไพ่ชุมชน(community cards), ไพ่ในมือ/ไพ่ปิด(hole cards).
- **핵심 명사**: card → **ไพ่**, hand(패) → **มือ / ไพ่ในมือ**, hand rankings → **ลำดับไพ่ / อันดับไพ่**, dealer → **ดีลเลอร์**, button → **ปุ่ม / บัตตอน (BTN)**, pot → **พอต / กองกลาง**, main pot → **พอตหลัก**, side pot → **ไซด์พอต / พอตข้าง**, kicker → **คิกเกอร์**, showdown → **โชว์ดาวน์ / เปิดไพ่**, suit → **ดอก**, stack → **สแตก / กองชิป**, chips → **ชิป**, Bluff → **บลัฟ**, ante → **แอนตี้**.
- **포지션 약어는 영어**(첫 등장 병기): UTG, CO(Cutoff), MP, HJ, SB, BB, BTN.

## 표기 (⚠️ **영어식 계열: en·zh·ar·ms와 동일 — 변환 없음**)
- **숫자 = 영어식 그대로**: 천단위 `,`(1,326·198,000 그대로), 소수점 `.`(2.5·2.7:1 그대로), % `35%`. **§13 값 표기 변환 불필요**(태국은 아라비아 숫자·영어식 구분자). 유럽식(1.326/2,5)으로 바꾸지 말 것.
- **카드 표기 라틴 유지**: `A♠`, `K♥`, `10♦`, 보드 그대로. 무늬 기호 ♠♥♦♣ 그대로.
- **화폐**: `$` 그대로(§13 보존 — ฿/บาท로 바꾸지 말 것).

## 문체
- **คุณ**체(정중·중립 2인칭, 태국 콘텐츠 표준) 또는 자연스러운 주어 생략. 태국 포커 커뮤니티(GGPoker th·ThaiPokerBeast)가 실제 쓰는 구어체(딱딱한 번역 티 금지). 태국어는 띄어쓰기 없이 자연스럽게.
- 태국 문자 성조부호·모음 표기 정확(สระ·วรรณยุกต์). 오타 = §13급 신뢰 리스크.
