# fil 포커 용어·표기 브리프 (번역 착수 전 웹리서치 결과)

> 대상 = **필리핀어/타갈로그(fil, Filipino/Tagalog)**. 필리핀(마닐라=아시아 포커 허브·PokerStars LIVE Manila 등). ★**중요**: 필리핀 포커 씬은 **영어가 절대 우세**(포커 전문 어휘가 전부 영어) + 일상 필리핀어 = **Taglish**(코드스위칭)가 자연스러운 실제 어법. 억지 타갈로그 신조어 금지(스와힐리와 유사). ★**직역 아님 — 현지 맥락 재구성.**

## 용어 (현지 관습 — Taglish: 필리핀어 산문 + 영어 포커용어)
- **핵심 원칙**: 포커 전문 용어(족보·액션·구조)는 **영어 그대로**(blinds, small/big blind, flop, turn, river, check, bet, call, raise, fold, all-in, pot, showdown, kicker, side pot, hand rankings, Royal Flush/Flush/Straight/Full House 등). 필리핀어 산문으로 설명·연결. Taglish가 실제 어법이니 영어 단어를 자연스럽게 섞어 씀.
- **족보 = 영어명 유지(첫 등장 시 필리핀어 설명)**:
  - Royal Flush → **Royal Flush** (pinakamataas na kamay)
  - Straight Flush → **Straight Flush**
  - Four of a Kind → **Four of a Kind** (apat na magkaparehong card)
  - Full House → **Full House**
  - Flush → **Flush** (limang card na iisang suit)
  - Straight → **Straight** (limang magkakasunod na card)
  - Three of a Kind → **Three of a Kind** (또는 trips)
  - Two Pair → **Two Pair**
  - One Pair → **Pair**
  - High Card → **High Card**
  - ★홀덤 순서 = **Full House > Flush**(표준, 변형 없음).
- **액션(영어 유지 + 필리핀어 산문)**: check → **check**, bet → **bet / tumaya**, call → **call**, raise → **raise / itaas**, fold → **fold / mag-fold**, all-in → **all-in**.
- **블라인드**: Blinds → **blinds**, Small Blind → **small blind (SB)**, Big Blind → **big blind (BB)**.
- **스트리트/구조**: preflop, flop, turn, river 그대로. betting round → **betting round / round ng taya**, community cards → **community cards**, hole cards → **hole cards**.
- **핵심 명사(필리핀어 or 영어)**: card → **card / baraha**, hand(패) → **hand / kamay**, hand rankings → **hand rankings**, dealer → **dealer**, button → **button (BTN)**, pot → **pot**, main pot → **main pot**, side pot → **side pot**, kicker → **kicker**, showdown → **showdown**, suit → **suit / palo**, stack → **stack**, chips → **chips**, table → **mesa / table**, player → **manlalaro / player**, money → **pera**, Bluff → **bluff**.
- **포지션 약어는 영어**(첫 등장 병기): UTG, CO(Cutoff), MP, HJ, SB, BB, BTN.

## 표기 (⚠️ **영어식 계열: en·zh·ar·ms·th·sw와 동일 — 변환 없음**)
- **숫자 = 영어식 그대로(미국식)**: 천단위 `,`(1,326·198,000 그대로), 소수점 `.`(2.5·2.7:1 그대로), % `35%`. **§13 값 변환 불필요**(필리핀 미국식 관습). 유럽식(1.326/2,5)으로 바꾸지 말 것.
- **카드 표기 라틴 유지**: `A♠`, `K♥`, `10♦`, 보드 그대로. 무늬 기호 ♠♥♦♣ 그대로.
- **화폐**: `$` 그대로(§13 보존 — ₱/PHP로 바꾸지 말 것).
