# he 포커 용어·표기 브리프 (번역 착수 전 웹리서치 결과)

> 대상 = **히브리어(he, עברית)**. 이스라엘 포커 글로시(pokertown.co.il · foxmind.co.il · gg7xl.com · 7xl.io · he.wikibooks 포커/דירוג_הידיים · mako.co.il) 기준. 이스라엘은 포커 씬 활발·용어 확립(음차+고유 혼용). ★**직역 아님 — 현지 맥락 재구성**. ★★**RTL(우→좌) 언어** — ar/fa RTL 인프라 사용.

## ⚠️ RTL·표기 특성 (중요)
- **히브리어는 RTL**(오른쪽→왼쪽). `lib/intl.ts` RTL_LOCALES에 "he" 추가됨. 카드 표기(A♠ K♥)·라틴 영어 용어·숫자는 LTR로 임베드(ar/fa 동일 처리, globals.css `[dir=rtl]` 재활용).
- **숫자 = 라틴/서양 숫자(0-9)**. 히브리 숫자(게마트리아 א/ב/ג) 쓰지 말 것 — 일반 숫자는 아라비아 숫자가 표준.

## 용어 (현지 관습 — 히브리어 음차+고유명)
- **족보 (히브리어명, 첫 등장 시 영어 병기)**:
  - Royal Flush → **רויאל פלאש**
  - Straight Flush → **סטרייט פלאש**
  - Four of a Kind → **רביעייה** (고유어 quads)
  - Full House → **פול האוס**
  - Flush → **פלאש**
  - Straight → **רצף** (고유어) 또는 **סטרייט**
  - Three of a Kind → **שלישייה** (고유어 trips) 또는 סט
  - Two Pair → **שני זוגות**
  - One Pair → **זוג**
  - High Card → **קלף גבוה**
  - ★홀덤 순서 = **פול האוס > פלאש**(표준, 변형 없음).
- **액션(히브리어 음차+동사)**: check → **צ'ק / לבדוק**, bet → **הימור / בט / להמר**, call → **קול / להשוות**, raise → **רייז / העלאה / להעלות**, fold → **פולד / לזרוק (לפרוש)**, all-in → **אול-אין**. 첫 등장 후 자연스러운 쪽.
- **블라인드**: Blinds → **בליינדים**, Small Blind → **סמול בליינד (SB)**, Big Blind → **ביג בליינד (BB)**. 영어 SB/BB 병기 가능.
- **스트리트/구조**: פרה-פלופ, פלופ, טרן, ריבר (히브리 음차). סבב הימורים(betting round), בורד(board), קלפי קהילה/קלפים משותפים(community cards), קלפים אישיים(hole cards).
- **핵심 명사**: card → **קלף**, hand(패) → **יד**, hand rankings → **דירוג ידיים**, dealer → **דילר**, button → **באטן / כפתור (BTN)**, pot → **קופה** (또는 פוט), main pot → **קופה ראשית**, side pot → **סייד פוט / קופה צדדית**, kicker → **קיקר**, showdown → **שואודאון / חשיפת קלפים**, suit → **סדרה / צורה**, stack → **סטאק / ערימה**, chips → **ז'יטונים / צ'יפים**, Bluff → **בלוף**, ante → **אנטה**.
- **포지션 약어는 영어**(첫 등장 병기): UTG, CO(Cutoff), MP, HJ, SB, BB, BTN.

## 표기 (⚠️ **영어식 계열: ar처럼 서양숫자·영어식 구분자**)
- **숫자 = 영어식 그대로**: 천단위 `,`(1,326·198,000 그대로), 소수점 `.`(2.5·2.7:1·0.84 그대로), % `35%`. **§13 값 표기 변환 불필요**(서양숫자·영어식). 히브리 숫자 금지.
- **카드 표기 라틴 유지**: `A♠`, `K♥`, `10♦`, 보드 그대로. 무늬 기호 ♠♥♦♣ 그대로(RTL 안에서 LTR 임베드).
- **화폐**: `$` 그대로(§13 보존 — ₪/שקל로 바꾸지 말 것).

## 문체
- **אתה**체(2인칭 남성 단수, 히브리 일반 콘텐츠 표준 — 성 중립 기본값). 자연스러운 이스라엘 포커 구어(딱딱한 문어체·기계번역 티 금지). 이스라엘 포커 커뮤니티가 실제 쓰는 표현·리듬.
- 히브리 문자·니쿠드 없이 표준 정서법. 철자 오타 = §13급 신뢰 리스크.
