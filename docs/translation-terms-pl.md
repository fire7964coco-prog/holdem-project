# pl 포커 용어·표기 브리프 (번역 착수 전 웹리서치 결과)

> 대상 = **폴란드어(pl, Polski)**. 폴란드 포커 글로시(pokerlistings.pl · pokerstrategy.pl/pl · ggpoker pl · pokertexas.net · zasady-pokera.pl · gamedesire.com/pl · sts.pl/blog) 기준. 폴란드는 **고유 포커 용어 확립**. ★**직역 아님 — 현지 맥락 재구성**.

## ⚠️⚠️ 치명 함정 (§13급 — 반드시 주의)
1. **Poker = Straight Flush!** 폴란드어에서 "poker"는 게임 이름이자 **스트레이트 플러시 족보명**(strit w kolorze). Royal Flush = **Poker królewski**. quads 아님!
2. **Kareta = Four of a Kind**(포카드).
3. **Kolor = Flush** (kolor=색/무늬도 됨 → 문맥 구분). flush 뜻일 때 족보 Kolor.
4. **Strit = Straight.**
5. **홀덤 순서 = Full > Kolor**(풀하우스 > 플러시, 폴란드 공식 hierarchy도 full > kolor로 홀덤 표준과 동일 ✓).
6. 확정 hierarchy: **Poker królewski > poker > kareta > full > kolor > strit > trójka > dwie pary > para > najwyższa karta.**

## 용어 (현지 관습)
- **족보 (폴란드어 표준명, 첫 등장 시 영어 병기)**:
  - Royal Flush → **Poker królewski**
  - Straight Flush → **Poker** ⚠️(strit w kolorze) — 첫 등장 "Poker (straight flush)"로 병기 권장(게임명 혼동 방지)
  - Four of a Kind → **Kareta**
  - Full House → **Full**
  - Flush → **Kolor** ⚠️(flush)
  - Straight → **Strit**
  - Three of a Kind / set → **Trójka** (set = set)
  - Two Pair → **Dwie pary**
  - One Pair → **Para**
  - High Card → **Najwyższa karta** (또는 wysoka karta)
- **액션**: check → **czekać / czek**, bet → **postawić / zakład**, call → **sprawdzić**, raise → **przebić / podbić / przebicie**, fold → **spasować / pas**, all-in → **all-in / va bank**. 첫 등장 후 자연스러운 쪽.
- **블라인드**: Blinds → **blindy / ciemne**, Small Blind → **mały blind / mała ciemna (SB)**, Big Blind → **duży blind / duża ciemna (BB)**. 영어 SB/BB 병기 흔함.
- **스트리트/구조**: preflop, flop, turn, river 그대로. runda betowania/licytacji(betting round), karty wspólne(community cards), karty zakryte(hole cards).
- **핵심 명사**: card → **karta**, hand(패) → **układ / ręka**, hand rankings → **układy / hierarchia układów**, dealer → **rozdający / dealer**, button → **button / przycisk (BTN)**, pot → **pula**, main pot → **główna pula**, side pot → **pula boczna / side pot**, kicker → **kicker**, showdown → **showdown / odkrycie kart / vskrycie**, suit → **kolor (masc)**, stack → **stack / stos żetonów**, chips → **żetony**, Bluff → **blef**, ante → **ante**.
- **포지션 약어는 영어**(첫 등장 병기): UTG, CO(Cutoff), MP, HJ, SB, BB, BTN.

## 표기 (⚠️ 유럽식 — 러시아/프랑스와 유사)
- **숫자 = 폴란드식**: 천단위 **공백**(spacja) → `1,326`→**`1 326`**, `198,000`→**`198 000`**, `1,300`→**`1 300`**. ⚠️마침표·콤마 천단위 금지. 소수점 **쉼표(,)** → `2.5`→**`2,5`**, `2.7:1`→**`2,7:1`**, `0.84`→**`0,84`**.
- **퍼센트**: `35%` 붙여쓰기 관습(소수 `0,84%`).
- **§13 값 자체는 불변, 구분자·공백만 폴란드식.** 라틴 숫자(0-9).
- **카드 표기 라틴 유지**: `A♠`, `K♥`, `10♦`, 보드 그대로. 무늬 기호 ♠♥♦♣ 그대로(족보 Kolor와 무늬는 문맥 구분).
- **화폐**: `$` 그대로(§13 보존 — zł/PLN로 바꾸지 말 것).
