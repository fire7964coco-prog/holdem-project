# fa 포커 용어·표기 브리프 (번역 착수 전 웹리서치 결과)

> 대상 = **페르시아어(fa, فارسی)**. 이란 포커 글로시(fa.wikipedia مقایسه_دست‌ها_در_پوکر · flopbaz.net · poker-baz.com · gopoker.global/fa · learn-poker.net · pokerestan.com) 기준. ★**직역 아님 — 현지 맥락 재구성**. ★★**RTL(우→좌) 언어** — ar과 같은 RTL 인프라 사용.

## ⚠️ RTL·표기 특성 (중요)
- **페르시아어는 RTL**(오른쪽→왼쪽). `lib/intl.ts` RTL_LOCALES에 "fa" 추가됨. 카드 표기(A♠ K♥)·라틴 영어 용어·숫자는 LTR로 임베드(ar과 동일 처리, globals.css `[dir=rtl]` 재활용).
- **숫자 = 라틴/서양 숫자(0-9)**. ⚠️**페르시아 숫자(۰۱۲۳۴۵۶۷۸۹) 쓰지 말 것** — §13 카드·수치 토큰 인식 유지 위해 ar과 동일하게 서양 숫자.

## 용어 (현지 관습)
- **족보 (페르시아어 음차+고유명, 첫 등장 시 영어 병기)**:
  - Royal Flush → **رویال فلاش**
  - Straight Flush → **استریت فلاش**
  - Four of a Kind → **کاره** (또는 چهارتایی)
  - Full House → **فول هاوس**
  - Flush → **فلاش** (또는 رنگ) ⚠️(رنگ=색/무늬와 다의어, 문맥 구분)
  - Straight → **استریت** (또는 ردیف)
  - Three of a Kind / set → **سه‌تایی** (또는 ست)
  - Two Pair → **دو جفت**
  - One Pair → **جفت** (또는 تک‌جفت)
  - High Card → **کارت بالا**
  - ★홀덤 순서 = **فول هاوس > فلاش**(풀하우스>플러시 표준, 이란 소스 동일 — 변형 없음).
- **액션(페르시아어 음차+동사)**: check → **چک / چک کردن**, bet → **شرط / بت**, call → **کال / همراهی کردن**, raise → **رِیز / افزایش دادن**, fold → **فولد / پاس / کنار گذاشتن**, all-in → **آل‌این**. 첫 등장 후 자연스러운 쪽.
- **블라인드**: Blinds → **بلایندها**, Small Blind → **بلایند کوچک (SB)**, Big Blind → **بلایند بزرگ (BB)**. 영어 SB/BB 병기 가능.
- **스트리트/구조**: پری‌فلاپ(preflop), فلاپ(flop), ترن(turn), ریور(river). دور شرط‌بندی(betting round), کارت‌های مشترک(community cards), کارت‌های بسته(hole cards).
- **핵심 명사**: card → **کارت**, hand(패) → **دست**, hand rankings → **رتبه‌بندی دست‌ها**, dealer → **دیلر**, button → **باتن / دکمه (BTN)**, pot → **پات**, main pot → **پات اصلی**, side pot → **ساید پات / پات جانبی**, kicker → **کیکر**, showdown → **شودان / رو کردن کارت‌ها**, suit → **خال**, stack → **استک**, chips → **چیپ‌ها / ژتون‌ها**, Bluff → **بلاف**, ante → **انته**.
- **포지션 약어는 영어**(첫 등장 병기): UTG, CO(Cutoff), MP, HJ, SB, BB, BTN.

## 표기 (⚠️ **영어식 계열: ar처럼 서양숫자·영어식 구분자**)
- **숫자 = 영어식 그대로**: 천단위 `,`(1,326·198,000 그대로), 소수점 `.`(2.5·2.7:1·0.84 그대로), % `35%`. **§13 값 표기 변환 불필요**(서양숫자·영어식). 페르시아/아랍 숫자(۰-۹, ٠-٩) 금지.
- **카드 표기 라틴 유지**: `A♠`, `K♥`, `10♦`, 보드 그대로. 무늬 기호 ♠♥♦♣ 그대로(RTL 안에서 LTR 임베드).
- **화폐**: `$` 그대로(§13 보존 — تومان/ریال로 바꾸지 말 것).

## 문체
- **شما**체(정중·표준 2인칭, 페르시아 웹 콘텐츠 관습). ⚠️유럽어들의 친근체(tu/du)와 달리 페르시아 문어체는 شما가 자연스럽고 표준 — تو는 너무 친밀해 부적절. 이란 포커 커뮤니티가 실제 쓰는 자연스러운 문어체(딱딱한 번역 티 금지).
- 페르시아 문자·زبان 정확(نیم‌فاصله ZWNJ 관습·حروف). 오타 = §13급 신뢰 리스크.
