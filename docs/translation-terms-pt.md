# pt-BR 포커 용어·표기 브리프 (번역 착수 전 웹리서치 결과)

> 대상 = **브라질 포르투갈어(pt-BR)**. 브라질 포커 커뮤니티/글로시(pokernobrasil·checkraise·GGPoker BR·partypoker BR·pokerlab) 기준.

## 용어 (현지 관습)
- **영어 그대로(가장 자연스러움)**: blinds, small blind / big blind, all-in, showdown, flop / turn / river, board, flush, kicker, fold(동사 "foldar"도 흔함), check(동사 "dar check"), pot odds, straddle, ante, heads-up, buy-in, stack, fish, bluff("blefe"도 통용).
- **포르투갈어로 옮김**: call → **pagar** / igualar, raise → **aumentar** / (subir), re-raise → aumentar de novo, pot → **pote**, side pot → **pote paralelo**, chips → **fichas**, suit → **naipe**, dealer → dealer/crupiê, split → dividir o pote.
- **브라질 속어**: all-in 하다 → **shovar**(shove 유래), 세트 적중 → trincar. (본문 톤에 맞춰 과하지 않게.)

## 표기
- **숫자**: 천단위 **점**(.), 소수점 **콤마**(,) → `1.326`, `198.000`, `0,003%`, `2,5 BB`, `2,7 para 1`. (es와 동일 함정 — 영어식 `1,326`·`2.5`는 오독.)
- **화폐**: 브라질은 R$지만 온라인 포커는 $가 일반 → §13 보존 위해 **$ 그대로**(환전 금지).
- 부호는 반각(? ! ( ))이 표준.
