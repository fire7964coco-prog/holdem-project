# uk 포커 용어·표기 브리프 (번역 착수 전 웹리서치 결과)

> 대상 = **우크라이나어(uk, Українська)**. 우크라이나 포커 글로시(ua.888poker.com · cardmates.ua · ggpoker.com/uk · poker.ua · visnyk.poker · uk.wikipedia Техаський_холдем) 기준. ★**중요**: 우크라이나어는 **러시아어와 별개 언어**(어휘·철자 다름) — ru를 복사하지 말 것. 우크라이나 고유 철자(і·ї·є·ґ·아포스트로프')로 저작. ★**직역 아님 — 현지 맥락 재구성.**

## 용어 (현지 관습 — 우크라이나어 음차)
- **족보 (우크라이나어 표준명 — 대부분 영어 음차, 첫 등장 시 영어 병기)**:
  - Royal Flush → **Флеш-рояль** (또는 Роял-флеш)
  - Straight Flush → **Стріт-флеш** (⚠️Стріт, ru의 Стрит 아님)
  - Four of a Kind → **Каре**
  - Full House → **Фул-хаус**
  - Flush → **Флеш**
  - Straight → **Стріт**
  - Three of a Kind / set → **Трійка** (또는 Сет)
  - Two Pair → **Дві пари**
  - One Pair → **Пара**
  - High Card → **Старша карта**
  - ★홀덤 순서 = **Фул-хаус > Флеш**(표준, 변형 없음).
- **액션(우크라이나어 음차+동사)**: check → **чек / чекати**, bet → **ставка / бет / ставити**, call → **колл / коллити / зрівняти**, raise → **рейз / рейзити / підвищити**, fold → **фолд / фолдити / скинути**, all-in → **олл-ін** (또는 ва-банк). 첫 등장 후 자연스러운 쪽.
- **블라인드**: Blinds → **блайнди**, Small Blind → **малий блайнд (SB)**, Big Blind → **великий блайнд (BB)**. 영어 SB/BB 병기 가능.
- **스트리트/구조**: префлоп, флоп, терн, рівер (우크라 음차). раунд торгівлі/торгів(betting round), борд(board), загальні карти(community cards), покет карти/кишенькові карти(hole cards).
- **핵심 명사**: card → **карта**, hand(패) → **рука** (또는 комбінація), hand rankings → **комбінації / рейтинг рук**, dealer → **дилер**, button → **баттон / кнопка (BTN)**, pot → **банк** (또는 пот), main pot → **основний банк**, side pot → **сайд-пот / побічний банк**, kicker → **кікер**, showdown → **розкриття / шоудаун**, suit → **масть**, stack → **стек**, chips → **фішки**, Bluff → **блеф**, ante → **анте**.
- **포지션 약어는 영어**(첫 등장 병기): UTG, CO(Cutoff), MP, HJ, SB, BB, BTN.

## 표기 (⚠️ 유럽식 계열 — 러시아와 유사)
- **숫자 = 우크라이나식**: 천단위 **공백**(пробіл) → `1,326`→**`1 326`**, `198,000`→**`198 000`**, `1,300`→**`1 300`**. ⚠️마침표·콤마 천단위 금지. 소수점 **쉼표(,)** → `2.5`→**`2,5`**, `2.7:1`→**`2,7:1`**, `0.84`→**`0,84`**.
- **퍼센트**: `35%` 붙여쓰기(소수 `0,84%`).
- **§13 값 자체는 불변, 구분자·공백만 우크라이나식.** 라틴 숫자(0-9) — 카드/수치 아라비아 숫자 그대로.
- **카드 표기 라틴 유지**: `A♠`, `K♥`, `10♦`, 보드 그대로. 무늬 기호 ♠♥♦♣ 그대로.
- **화폐**: `$` 그대로(§13 보존 — ₴/грн로 바꾸지 말 것).

## 문체
- **ти**체(친근 2인칭). 사이트 타 언어 친근체와 일관. 우크라이나 포커 커뮤니티·스트리머가 실제 쓰는 자연스러운 구어(딱딱한 관공서식·기계번역 티 금지). ★러시아어 아닌 순수 우크라이나어(і/ї/є/ґ·아포스트로프 п'ять 등).
- 우크라이나 문법·격변화(відмінки)·정확한 철자. 오타·러시아어 혼입 = §13급 신뢰 리스크.
